# assignment operator
[JS FILE](./9-assignment-operator.md)
> table of assingnment operator

| operator | example | same as|
|-------------|-------------|---|
|=|x = y|x=y|
|+=|x += y|x = x + y|
|-=|x -= y|x = x - y|
|*=|x *= y|x = x * y|
|/=|x /= y|x = x / y|
|%=|x %= y|x = x % y|
|**=|x **= y|x = x ** y|
> we can use assingnent opertor as a compare and give some value

var b = 10
a = b
document.write(a)
